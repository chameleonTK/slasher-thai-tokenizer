# What is It? #

Maximum munch tokenizer for Thai Language with a rule-based extension to merge. 

### Our Target ###

* Correctness [up to 90%]  
* Easy to apply
* Expandable
* Support Real time tokenize


### Install ###

```bash
$ git clone https://kramatk@bitbucket.org/kramatk/slasher.git
$ npm install
$ node app.js

```
